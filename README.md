# Werwolf Moderator — Web-Version

Diese Web-App ist ein Offline-fähiges Moderator-Tool für das Werwolf-Spiel (auf Deutsch).

Features:
- Spieler hinzufügen/entfernen
- Rollen konfigurieren und zufällig zuweisen
- Nacht-Flow mit Rüstung, Dieb, Bäcker, Seher, Dorfmatratze, Werwölfe, Hexe
- Tagphase mit Lynch (Dorftrottel gewinnt bei Lynch)
- Log und Download-Funktion
- Speichert Zustand lokal (localStorage)
- Dark Mode unterstützt via prefers-color-scheme

Nutzung:
1. Lade die Dateien als GitHub-Repository hoch (alle Dateien im Ordner).
2. Öffne index.html im Browser (oder hoste via GitHub Pages).
3. Als Spielleiter: nur du bedienst die Seite.

Hinweis: Diese App ist als Moderator-Hilfsmittel gedacht und nicht spielerverteilend.
